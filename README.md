# App Mòbil Nativa amb IA Local (TensorFlow.js)

Aquesta és una aplicació mòbil híbrida desenvolupada amb **Ionic, Vue.js i Capacitor** que utilitza **TensorFlow.js** per realitzar detecció d'objectes en temps real directament al dispositiu.

Projecte realitzat per a la pràctica de DAM.

## 🚀 Característiques
- **Detecció Local**: Utilitza el model COCO-SSD per identificar objectes sense connexió a Internet.
- **Rendiment Fluid**: Optimitzat amb acceleració WebGL.
- **Interfície Prèmium**: Disseny modern en mode fosc adaptat a dispositius mòbils.
- **Arquitectura Spec-Driven**: Desenvolupat seguint les fases de Foundations, Specify i Planning.

## 🛠️ Stack Tecnològic
- **Framework**: Ionic + Vue.js 3
- **Motor d'IA**: TensorFlow.js
- **Model**: COCO-SSD
- **Bridge Natiu**: Capacitor

## 📦 Instal·lació
1. `npm install`
2. `npm run build`
3. `npx cap sync android`
4. `npx cap open android`
